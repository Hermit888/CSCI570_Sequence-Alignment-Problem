# CSCI570 Sequence Alignment Problem

This repository contains the CSCI 570 sequence alignment project materials.

## Memory-Efficient Part - Yichen Yang (4403-8375-34)

`efficient.py` implements the memory-efficient alignment algorithm using dynamic
programming plus divide-and-conquer (Hirschberg-style recursion).

It only handles the alignment stage and assumes the two full input strings have
already been generated by another component.

## Run

```bash
bash efficient.sh <input_file> <output_file>
```

Or directly:

```bash
python3 efficient.py <input_file> <output_file>
```

## Output Format

The output file contains:

1. alignment cost
2. complete aligned first string
3. complete aligned second string
4. running time in milliseconds
5. memory usage in KB

## Input Format

The input file should contain exactly two generated strings:

1. full string X
2. full string Y

## Teammate Interface

The preprocessing/string-generation part should hand `efficient.py` an input
file with exactly two lines:

1. the fully generated first sequence
2. the fully generated second sequence

Run:

```bash
bash efficient.sh <generated_input_file> <output_file>
```

`efficient.py` will write:

1. optimal alignment cost
2. complete aligned sequence X
3. complete aligned sequence Y
4. running time in milliseconds
5. memory usage in KB
